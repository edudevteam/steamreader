- Use Gemini CLI to build a website
- Grow in your understanding of Prompt Engineering
- Connect a GitHub Repo to Cloudflare Pages
- Get a live site up for FREE.

## ✅ Prerequisites

- Git, GitHub, and GitHub Desktop
- Cloudflare Account Creation
- VSCode IDE (not Visual Studio)
