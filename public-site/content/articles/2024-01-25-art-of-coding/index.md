# Where Art Meets Technology

Programming isn't just about numbers and logic - it's a creative tool for making art! From animated movies to video games, code brings imagination to life.

## Why Learn Creative Coding?

- Express your creativity in new ways
- Make interactive art that responds to viewers
- Create animations and games
- Combine art with technology skills

## Getting Started with Scratch

Scratch is a free programming language designed for beginners. It uses colorful blocks that snap together like puzzle pieces.

### Your First Animation

Let's make a dancing cat!

1. Go to [scratch.mit.edu](https://scratch.mit.edu)
2. Start a new project
3. Use these blocks:

```
when green flag clicked
forever
  next costume
  wait 0.2 seconds
end
```

## Digital Art with Code

### Drawing Shapes

In many programming languages, you can draw with code:

```python
import turtle

# Draw a colorful spiral
colors = ['red', 'orange', 'yellow', 'green', 'blue', 'purple']

for i in range(100):
    turtle.color(colors[i % 6])
    turtle.forward(i * 2)
    turtle.right(61)
```

### Generative Art

Generative art uses algorithms to create unique pieces. Each time you run the code, you get something different!

Key concepts:
- **Randomness** - Adding unpredictable elements
- **Loops** - Repeating patterns
- **Colors** - Using math to create gradients

## Tips for Creative Coding

1. **Start simple** - Complex art often comes from simple rules
2. **Embrace mistakes** - Bugs can lead to beautiful accidents
3. **Share your work** - Get feedback from others
4. **Find inspiration** - Look at nature, architecture, and other art
5. **Keep experimenting** - There's no wrong way to make art
